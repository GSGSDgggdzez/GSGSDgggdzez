Laravel Project
This is a Laravel project that includes various features and functionalities. It utilizes jQuery, AJAX, and the RESTful Countries API to fetch and display country and state information.

Getting Started
To get started with this project, follow these steps:

 -Prerequisites
 -PHP (version 7.4 or higher)
 -Composer
 -Node.js (version 12 or higher)
 -npm

 Installation
1-download ZIP¨file:

2-Navigate to the project directory:

3-Install the PHP dependencies:
                       
          composer install
4-Install the JavaScript dependencies:

           npm install
5-Create a copy of the .env.example file and rename it to .env. Then, generate an application key:

     cp .env.example .env
     php artisan key:generate
6-Configure the database connection in the .env file.

7-Run the database migrations:

    php artisan migrate
    
8-Build the assets

    npm run dev
9-Start the development server:

    php artisan serve

The application should now be running at http://localhost:8000.

USAGE
The project includes the following features:

-Fetching a list of countries from the RESTful Countries API and displaying them in a dropdown.
-Fetching a list of states for a selected country from the RESTful Countries API and displaying them in a dropdown.
-Using jQuery and AJAX to make API requests and update the page dynamically.
-Implementing sticky navigation using the jQuery Sticky plugin.
-Utilizing Laravel's built-in features like routing, controllers, and views.
-You can explore the codebase to understand how these features are implemented and make modifications as needed.
